☁️ Sincronização automática (nuvem do Google)

Com a sincronização ativa, tudo o que você lançar em um aparelho aparece nos outros automaticamente. Os dados ficam num arquivo do seu Google Drive. Sem internet, o sistema salva no aparelho e sincroniza quando conectar.

📖 Como configurar (uma única vez, ~10 minutos)
  1. No computador, acesse script.google.com logado no seu Gmail e clique em Novo projeto.
  2. Apague o conteúdo do editor e cole o código abaixo. Na primeira linha, troque mude-esta-senha por uma senha sua.
  3. Clique em Implantar → Nova implantação → engrenagem → App da Web. Em "Executar como": Eu. Em "Quem pode acessar": Qualquer pessoa. Clique Implantar e autorize com sua conta.
  4. Copie a URL do app da Web (termina em /exec), cole aqui em cima junto com a senha e clique Ativar sincronização.
  5. Repita só o passo 4 nos outros aparelhos (mesma URL e senha).

⚠️ Guarde a URL e a senha só com você: quem tiver as duas consegue ler e alterar os dados.

const SENHA = 'mude-esta-senha';
const NOME_ARQUIVO = 'financas-igreja-pessoal.json';
function doGet(e){ return responder(e); }
function doPost(e){ return responder(e); }
function responder(e){
  const out = o => ContentService.createTextOutput(JSON.stringify(o)).setMimeType(ContentService.MimeType.JSON);
  if (!e || !e.parameter || e.parameter.k !== SENHA) return out({ok:false, error:'senha incorreta'});
  const it = DriveApp.getFilesByName(NOME_ARQUIVO);
  let f = it.hasNext() ? it.next() : null;
  if (e.postData && e.postData.contents){
    if (f) f.setContent(e.postData.contents);
    else f = DriveApp.createFile(NOME_ARQUIVO, e.postData.contents, 'application/json');
    return out({ok:true});
  }
  return ContentService.createTextOutput(f ? f.getBlob().getDataAsString() : 'null')
    .setMimeType(ContentService.MimeType.JSON);
}

Backup manual

Mesmo com a sincronização ativa, vale exportar um backup de vez em quando como segurança extra. Sem sincronização, é assim que você leva os dados de um aparelho a outro.

Configurações

As senhas protegem contra olhares casuais (abertura do sistema, painel Pessoal, relatórios pessoais, movimentos e exportações). Os dados continuam gravados no aparelho — não é criptografia.
